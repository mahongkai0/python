　　　变量
　　　　　什么是变量
　　　　　　　变量是关联一个对象的标识符
　　　　　　　变量可以绑定一个对象，并可以通过变量名使用这个对系
　　　　
　　　　标识符的命名方法
　　　　　　以字母或下划线开头，后跟字母下滑线或数字
　　　　　说明：
　　　　　　　　　标识符区分大小写
　　　　　　示例：
　　　　　　　　ａ　A a1 bbbb set_age _ _abcd _abcde a1b2
   不合法的标识符
   　　　1ａ　123　$ABC 
   python的关键字不能用作变量名
　　　　　True False None is del if

       赋值语句　statemrnts
         符号　＝
         语法：
         　　变量名　＝　表达式
         　　或
         　　变量名1　= 变量名2 ＝　表达式 
            或
　          变量名1, 变量名2,  ... = 序列
　　　　　作用：
　　　　创建一个变量并绑定
　　　　改变一个变量的绑定的数据对象
　　　　示例：
　　　　　　　a = 10
            b =20
            c = a + b # c 绑定　30 
        a = b = c = 10000  #同时改变三个变量绑定ｕ关系
　　　　　　　　　　　　　　　　＃　同时绑定在10000上
　ｘ,y = 100, 200    #用x绑定100　　用y绑定200

　赋值语句说明：
　　　1.当变量不存在时，赋值语句会创建该变量，同时将变量
　　　　绑定在赋值运算符右侧的对象上
　　　2.当变量存在时，改变此变量的绑定关系
　　　3.一个变量只能绑定一个对象
　　　4.多个变量可以同时绑在一个对象上
   del 语句
   作用：
   　　用于删除变量，同时解除与对象的关联关系，如果可能则释放对象
   语法：
   　　　del 变量名1，变量名2，....

在交互模式下查看当前作用域内的所有变量：

　自动化内存管理和引用计数 :
    每个对象都会记录有几个变量引用（绑定）自身，当引用的数量为
    0时，则有些对象被销毁，此种自动化内存管理的方式叫
    ＂引用计数＂

　　is 和 is not 运算符
　　　　作用：
　　　　　　判断两个对象是否是同一个对象，当是同一个对象时
　　　　返回 True,否则返回　False
      
         is not 与is　的返回值相反
         　　　语法：
         　　x is y
            x is not y　　　　　　实质是判断两个对象的id是否相同
id 函数：
　　　作用：
　　　　　　返回一个对象在内存中的地址
　　　说明:
         is　运算符是根据id来进行判断的
